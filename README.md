# math_lib
- 그래픽 렌더링에 필요한 최소한의 벡터 및 행렬 연산을 도와주는 라이브러리
    - 이미 만들어진 좋은 라이브러리들이 있지만 (glm, boost, eigen 등등) 선형 대수 학습하면서 직접 구현
    - 해당 라이브러리를 나만의 렌더러 만들기 프로제크에서 사용 예정
- 구현 내용
    - 벡터
        - vec2, vec3, vec4
        - 연산
            - 기본 사칙연산
            - 비교 연산 (동등)
            - 내적 외적
    - 행렬
        - 정방행렬
            - mat2, mat3, mat4
        - 연산
            - 기본 사칙연산
            - 비교 연산 (동등)
            - 전치 행렬, 역행렬
